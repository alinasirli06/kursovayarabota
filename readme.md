# АТЛАНТ — Информационно-справочная система для фитнес-центра

## О проекте

Данный проект представляет собой современную информационно-справочную систему для фитнес-центра «АТЛАНТ». Сайт разработан с использованием HTML, CSS и JavaScript и предназначен для предоставления пользователям полной информации о фитнес-центре, услугах, расписании занятий и способах связи.

Основная цель проекта — создание удобного и понятного веб-интерфейса для клиентов фитнес-центра.

---

# Использованные технологии

## HTML5
HTML использовался для создания структуры всех страниц сайта, размещения контента, навигации, таблиц, форм и других элементов интерфейса.

## CSS3
CSS использовался для:
- стилизации страниц;
- создания адаптивного дизайна;
- анимаций;
- оформления кнопок, карточек и форм;
- создания современного интерфейса.

## JavaScript
JavaScript использовался для:
- открытия и закрытия модального окна;
- работы мобильного меню;
- валидации форм;
- кнопки прокрутки вверх;
- интерактивности интерфейса.
